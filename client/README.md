# Won Games

### Tecnologies
- React
- NextJS
- Styled Components
- Storybook
- Typescript

### Local development
```bash
# Start aplication
$ yarn dev

# Start Storybook
$ yarn storybook
```
### Storybook
```bash
# Build Storybook
$ yarn build-storybook

# Start Storybook
$ yarn storybook
```
### Build Production
```bash
# Build command
$ yarn build
```
